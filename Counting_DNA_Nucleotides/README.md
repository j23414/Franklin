**Perl**

```
$ cat DATA/sample.txt | ./countLetters.pl

	1
A	20
C	12
G	17
T	21
```

**Python**

```
$ cat DATA/sample.txt | ./countLetters.py

	1
A 	20
C 	12
G 	17
T 	21
```

**Rscript**

```
$ cat DATA/sample.txt | ./countLetters.R
```
Instead of printing out counts, Rscript will generate a **barchart.png** of Letter Frequencies
